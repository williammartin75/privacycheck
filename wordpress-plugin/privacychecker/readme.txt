== Description ==

1. Upload the plugin files to `/wp-content/plugins/privacychecker` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings â†’ PrivacyChecker to configure the plugin

== Frequently Asked Questions ==

= Do I need a PrivacyChecker account? =

Yes, you need a Site ID from [privacychecker.pro](https://privacychecker.pro) to use this plugin. You can get a free account.

= Is this plugin GDPR compliant? =

Yes! PrivacyChecker is designed for GDPR compliance with:
* Prior consent before non-essential cookies
* Granular consent options (analytics, marketing, functional)
* Easy withdrawal of consent
* Audit trail of all consents

= Does it work with Google Analytics 4? =

Yes! PrivacyChecker integrates with Google Consent Mode v2, which works with GA4 and Google Ads.

= Can I customize the banner appearance? =

Yes, you can customize:
* Primary button color
* Banner position (top or bottom)
* Language
* Privacy policy link

= Does it slow down my site? =

No, the banner script is lightweight and loaded asynchronously. It has minimal impact on page speed.
